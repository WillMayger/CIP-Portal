# CIP-Portal
A repository for the CIP Portal. 

## Folder Structure:
~~~
src/client -> flux + react 
src/server -> backend
~~~
### Folder Structure - Tests:
~~~
__tests__ -> ComponentName-test.js -> UnitTesting using react + jest
~~~
## Use:
~~~
1. git clone https://github.com/CanopyCloud/CIP-Portal
2. cd CIP-Portal 
3. npm install
4. npm start
~~~

## Run Tests:
~~~
1. npm install
2. gulp test
~~~
